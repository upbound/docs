customer-hosted deployments of the Upbound Spaces software, including
Self-Hosted Spaces (customer-managed) and Managed Spaces (Upbound-managed,
customer-hosted).

If you use Upbound's hosted SaaS offering, see the
[Cloud Spaces documentation](/cloud-spaces/overview/).
