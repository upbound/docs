# Upbound Cloud Spaces

Welcome to the Upbound Cloud Spaces documentation. Cloud Spaces are Upbound-hosted,
Upbound-managed Space environments that provide a SaaS experience for running
Crossplane control planes.

## Deployment models

Cloud Spaces come in two flavors:

- **[Cloud Spaces][cloud-spaces]**: Multi-tenant Upbound-hosted, Upbound-managed Space environment.
- **[Dedicated Spaces][dedicated-spaces]**: Single-tenant Upbound-hosted, Upbound-managed Space environment with additional isolation guarantees.
