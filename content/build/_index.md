Authoring and building your control plane projects is the best way to get hands-on and experience the value of Upbound.

In this guide, you'll learn how to
- Instantiate a control plane project and add dependencies
- Author your first API
- Author your first composition and composition function
- Build and Push your control plane project
- Run and Test your control plane project
