Upbound Crossplane supports running [Official Packages][official-packages] released by Upbound. These packages extend your Upbound Crossplane cluster with new capabilities, user-defined logic, and add-ons. 

## Package availability

**Official Packages that are compatible with Crossplane v2.0 are Upbound-Crossplane only.** Patch version releases of Official Packages require a commercial license, available for users on a Standard plan or greater.

## Learn about Packages

Read the [package concept][package-concept] documentation to learn how packages work in Upbound Crossplane. Read the Official Package [policies][policies] documentation to learn about how to access different packages and versions on Upbound Crossplane.
