Get a token for the robot.

#### Usage

```bash
up robot token get <robot-name> <token-name>
```

#### Arguments

##### `<robot-name>`

**Required**

Name of robot.

##### `<token-name>`

**Required**

Name of token.

#### Examples

```bash
# Show help
up robot token get --help

# Basic usage
up robot token get <robot-name> <token-name>
```
