Interact with robot tokens.

#### Usage

```bash
up robot token [subcommand]
```

#### Subcommands

- `create` - Create a token for the robot.
- `delete` - Delete a token for the robot.
- `list` - List the tokens for the robot.
- `get` - Get a token for the robot.

#### Examples

```bash
# Show help
up robot token --help
```
