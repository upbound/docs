Install UXP.

#### Usage

```bash
up uxp install [version] [flags]
```

#### Arguments

##### `<version>`

**Optional**

UXP version to install.

#### Flags

##### `--unstable`

Allow installing unstable versions.

##### `--set`

Set parameters.

##### `--file` / `-f`

Parameters file.

##### `--bundle`

Local bundle path.

#### Examples

```bash
# Show help
up uxp install --help

# Basic usage
up uxp install <version>
```
