Alpha features. Commands may be removed in future releases.

#### Usage

```bash
up alpha [subcommand]
```

#### Subcommands

- `migration` - Migrate control planes to Upbound Managed Control Planes.
- `space` - Interact with Spaces.
- `ctx` - Select an Upbound kubeconfig context.

#### Examples

```bash
# Show help
up alpha --help
```
