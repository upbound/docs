This guide contains the best practices and foundations for building with
Upbound. In this guide, you'll create a bootstrapped Upbound Space environment
with AWS and a GitOps integration.

## Prerequisites

To create your platform foundations, you'll need:

* an Upbound account
* an AWS account
* `kubectl`
* the Upbound CLI
