Delete a group.



#### Usage

`up group delete <name> [flags]`
#### Arguments

| Argument | Description |
| -------- | ----------- |
| `<name>` | Name of group. |
#### Flags

| Flag | Short Form | Description |
| ---- | ---------- | ----------- |
| `--force` | | Force the deletion of the group. |
