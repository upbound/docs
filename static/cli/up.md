The Upbound CLI.

Please report issues and feature requests at https://github.com/upbound/upbound.

#### Options

##### `--help`
*Shorthand:* `-h`  
Show context-sensitive help.

##### `--format`
*Default:* `default`  
Format for get/list commands. Can be: json, yaml, default

##### `--quiet`
*Shorthand:* `-q`  
Suppress all output.

##### `--pretty`
Pretty print output.

##### `--dry-run`
dry-run output.
