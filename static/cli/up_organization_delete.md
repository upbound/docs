Delete an organization.



#### Usage

`up organization delete <name> [flags]`
#### Arguments

| Argument | Description |
| -------- | ----------- |
| `<name>` | Name of organization. |
#### Flags

| Flag | Short Form | Description |
| ---- | ---------- | ----------- |
| `--force` | | Force deletion of the organization. |
