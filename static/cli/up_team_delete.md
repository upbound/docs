Delete a team.



#### Usage

`up team delete <name> [flags]`
#### Arguments

| Argument | Description |
| -------- | ----------- |
| `<name>` | Name of team. |
#### Flags

| Flag | Short Form | Description |
| ---- | ---------- | ----------- |
| `--force` | | Force delete team even if conflicts exist. |
