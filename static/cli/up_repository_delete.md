Delete a repository.



#### Usage

`up repository delete <name> [flags]`
#### Arguments

| Argument | Description |
| -------- | ----------- |
| `<name>` | Name of repository. |
#### Flags

| Flag | Short Form | Description |
| ---- | ---------- | ----------- |
| `--force` | | Force deletion of repository. |
