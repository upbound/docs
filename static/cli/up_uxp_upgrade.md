Upgrade UXP.

#### Options

##### `--rollback`
Rollback to previously installed version on failed upgrade.

##### `--force`
Force upgrade even if versions are incompatible.

##### `--unstable`
Allow installing unstable versions.

##### `--set`
Set parameters.

##### `--file`
*Shorthand:* `-f`  
Parameters file.

##### `--bundle`
Local bundle path.
