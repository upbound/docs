Delete a robot.



#### Usage

`up robot delete <name> [flags]`
#### Arguments

| Argument | Description |
| -------- | ----------- |
| `<name>` | Name of robot. |
#### Flags

| Flag | Short Form | Description |
| ---- | ---------- | ----------- |
| `--force` | | Force delete robot even if conflicts exist. |
