Install a AddOn.



#### Usage

`up controlplane add-on install <package> [flags]`
#### Arguments

| Argument | Description |
| -------- | ----------- |
| `<package>` | Reference to the AddOn. |
#### Flags

| Flag | Short Form | Description |
| ---- | ---------- | ----------- |
| `--name` | | Name of AddOn. |
| `--package-pull-secrets` | | List of secrets used to pull AddOn. |
| `--wait` | `-w` | Wait duration for successful AddOn installation. |
