Open the UXP web UI.



#### Usage

`up uxp web-ui open [flags]`
#### Flags

| Flag | Short Form | Description |
| ---- | ---------- | ----------- |
| `--host` | | Host to listen on for port-forward. |
| `--port` | | Port to listen on for port-forward (0 for automatic selection). |
| `--browser` | | Open the web UI in a browser window. |
