Create a robot.



#### Usage

`up robot create <name> [flags]`
#### Arguments

| Argument | Description |
| -------- | ----------- |
| `<name>` | Name of robot. |
#### Flags

| Flag | Short Form | Description |
| ---- | ---------- | ----------- |
| `--description` | | Description of robot. |
